# nest-vite-node-starter

Nest.JS via vite example

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov

# e2e tests
$ npm run test:e2e
```
